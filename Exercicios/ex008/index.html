Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usanto a tag <b> (não semântico)

Nesta frase temos um termo em negrito usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica)

Nesta frase temos um termo em itálico usando a tag <em> (semântica)

Texto marcado

Podemos criar támbem um texto marcado usando a tag <mark> (ctrl shift p)

e no outro parágrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.Usando a tag <big> e a tag <small>

Texto deletado

podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma Ênfase e indicar que ele foi adicionado depois usando a tag <ins>

Podemos marcar um texto como inserido para dar uma Ênfase e indicar que ele foi adicionado depois usando a tag <u> (não semantica)

Texto sobrescrito

Para inserir coisas do tipo x2+3 usando a tag <sup>

Texto subscrito

Para inserir coisas tipo H2O usando a tag <sub>
